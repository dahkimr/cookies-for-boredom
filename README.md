# cookies-for-boredom-react

Website where randomized activities are given when users click on the jar (built in React and MongoDB).

Run Instructions
----------------
Frontend:  
go into client folder  
run npm start

Backend:  
go into api folder  
run npm start

run mongodb:  
go to C:\Program Files\MongoDB\Server\4.2\bin   
run mongod
